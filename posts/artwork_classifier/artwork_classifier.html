Artwork Single-Label Classifer Using Fastai

I’ve always been interested in how AI/ML can apply to visual arts. In some ways, computer vision applications feel like the most direct application to art since we can build models to classify, modify and even generate visuals in an artistic fashion!

I am still very early in my deep learning studies but it’s exciting to tinker around with the deep learning tools and concepts I have in an art context. In this notebook, I will be exploring how to develop image classification models that can classify artwork based on three types: pencil drawings, paintings, and sculptures. My hope is for this notebook to become the start of projects that help other deep learning practitioners along their own journeys!

Inspiration, Toolkit, Data

The inspiration for this notebook comes from the computer vision lessons from fastai, a deep learning framework built mainly over Pytorch. I’m continuing to use fastai for a few reasons:

  • The capability to train modern, well-tuned models right out of the box
  • High-level abstractions that can cooperate with low-level capabilities
  • Focus on architecture and prototyping rather than fine details

For the image data, I am collecting Bing images of the artwork using search terms for the three types of art I am studying.

With a bit of that context out of the way, we can get started!

def get_dls(bs: int, size: int) -> DataLoaders:
    db = DataBlock(
        blocks=(ImageBlock(), CategoryBlock()),
        get_items=get_image_files,
        splitter=RandomSplitter(0.2, seed=123),
        get_y=parent_label,
        item_tfms=Resize(460),
        batch_tfms=aug_transforms(size=size, min_scale=0.75)
    )
    return db.dataloaders(path, bs=bs)

for bs in [32, 40, 50, 64, 128]:
    dls = get_dls(bs, 256)

        dls, resnet34, metrics=accuracy, cbs=[MixedPrecision])
    lr_steep = learn.lr_find(suggest_funcs=steep, show_plot=False)
    learn.fit_one_cycle(5, lr_steep)
    _, acc = learn.validate()
    bs_accs[bs] = acc
